# 📝 **Projeto para Blog colaborativo**

Bem-vindo ao sistema de gerenciamento para Blog colaborativo! Este projeto utiliza **Laravel** no backend e **Next** no frontend.

## 🌟 **Funcionalidades**
- Gerenciamento completo de Blog.
- Interface moderna e responsiva.
- Backend robusto e seguro com Laravel.
- Frontend dinâmico e modular com Next.

---

## 🛠️ **Tecnologias Utilizadas**
- **Backend:** Laravel 11.51.0
- **Frontend:** Next 16.2.4
- **Gerenciamento de Pacotes:** NPM
- **Outras:** Firebase (em modo desenvolvimento)
